# ai-spatial-movement-workflow
Dieses Projekt zeigt einen KI-gestützten Workflow zur Erfassung, Analyse und Prognose räumlicher Bewegungsmuster. Mithilfe von Computer Vision werden Laufpfade erkannt, Heatmaps erstellt und Bewegungsverteilungen prognostiziert, um räumliche Planung und Layoutentscheidungen zu unterstützen.
